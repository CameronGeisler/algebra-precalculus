    Section 3.10 Quadratic Inequalities in One Variable

    In addition to quadratic equations, we can also consider quadratic inequalities, which are closely related.
    🔗

    Example 3.10.1. Motivational Example: Solving with a Graph.

    Consider the inequality,
    🔗
    \begin{equation*} x^2-4x+3 \gt 0 \end{equation*}
    The goal is to find all of the values of \(x\) such that \(x^2 - 4x + 3 \gt 0\text{.}\) In simple terms, this means all numbers \(x\text{,}\) where if you substitute it into \(x^2-4x+3\text{,}\) the result is positive. This is most intuitive by visualizing the graph of the quadratic function \(y = x^2 - 4x + 3\text{,}\)
    🔗
    Then, determining where \(x^2-4x+3 \gt 0\) is equivalent to determining the \(x\) where the \(y\)-coordinate of the parabola is positive. Observe that the \(x\)-intercepts of the graph are \(x = 1, 3\text{.}\) So, the graph is positive when \(x \lt 1\) or when \(x \gt 3\text{.}\) Therefore, the solution is \(x \lt 1\) or \(x \gt 3\text{.}\)
    🔗
    🔗

    Example 3.10.2. Motivation: sign analysis with roots.

    Consider the inequality \(x^2-x-6 \lt 0\text{.}\) Like before, first, find the roots of the parabola, by making it an equals sign, and factoring,
    🔗
    \begin{align*} x^2 - x - 6 \amp= 0\\ (x-3)(x+2) \amp= 0 \end{align*}
    So, the roots are \(x = 3\) and \(x = -2\text{.}\) Since \(x^2-x-6\) has a positive leading coefficient (+1), the parabola opens up. This means that the quadratic roughly looks like this,
    🔗
    From this, the parabola is negative when \(x\) is between \(-2\) and \(3\text{,}\) or \(-2 \lt x \lt 3\text{.}\)
    🔗
    🔗
    To solve the inequality, we don’t need to graph the entire parabola precisely. Instead, we only need to find the roots, and know the basic shape (opening up or down).
    🔗
    A quadratic expression can only change sign at its roots. This means, if a quadratic has two roots, then:
    🔗
    • If the quadratic is positive in one interval, it will switch to negative when crossing a root.
      🔗
      🔗
    • After crossing the second root, it switches back to positive.
      🔗
      🔗
    If the quadratic opens down, then the pattern is reversed: it first is negative, then positive, then negative. The pattern of signs follows,
    🔗
    \begin{equation*} \text{either} \quad + \, - \, + \quad \text{or} \quad - \, + \, - \end{equation*}

    Subsection 3.10.2 Solving Quadratic Inequalities Summary

    Definition 3.10.4.

    A quadratic inequality is an inequality that can be written in one of the forms,
    🔗
    \begin{align*} ax^2 + bx + c \amp\leq 0\\ ax^2 + bx + c \amp\lt 0\\ ax^2 + bx + c \amp\geq 0\\ ax^2 + bx + c \amp\gt 0 \end{align*}
    🔗
    1. (If necessary) Arrange the quadratic inequality in standard form, with \(ax^2 + bx + c\) on one side, and 0 on the other.
      🔗
      🔗
    2. Find the critical values (the roots of the quadratic expression), by solving the corresponding quadratic equation \(ax^2 + bx + c = 0\text{.}\)
      🔗
      🔗
    3. Sketch a number line, and mark the critical values to split it into 3 intervals.
      🔗
      🔗
    4. Find where the quadratic is positive or negative
      🔗
      • If it opens up (\(a \gt 0\)), the sign pattern is \(+ \, - \, +\text{.}\)
        🔗
        🔗
      • If it opens down (\(a \lt 0\)), the sign pattern is \(- \, + \, -\text{.}\)
        🔗
        🔗
      🔗
    5. Choose the intervals that match the inequality symbol (greater than or less than).
      🔗
      🔗
    6. Include the endpoints if the inequality is \(\geq\) or \(\leq\) (non-strict inequality). Don’t include the endpoints if it’s \(\gt\) or \(\lt\text{.}\)
      🔗
      🔗
    🔗

    Subsection 3.10.3 Solving Using Sign Chart and Test Points

    There is another method to solve a quadratic inequality, that is more systematic and general, but less efficient for quadratic inequalities, called the sign chart method, or test point method.
    🔗

    Example 3.10.5.

    Consider the inequality \(x^2 - 6x + 8 \gt 0\text{.}\) Similar to before, first find the critical values, by factoring,
    🔗
    \begin{align*} x^2 - 6x + 8 \amp= 0\\ (x-2)(x-4) \amp= 0 \end{align*}
    So, the critical values are \(x = 2\) and \(x = 4\text{.}\) Set up a number line, with these roots marked,
    🔗
    These roots split up the number line into 3 intervals: \(x \lt 2\text{,}\) \(2 \lt x \lt 4\text{,}\) and \(x \gt 4\text{.}\) For each of these intervals, the quadratic will either be positive everywhere, or negative everywhere. So, it is enough to test a single point, and see if the quadratic is positive or negative. For example,
    🔗
    • For \(x \lt 2\text{,}\) choose \(x = 1\text{:}\)
      🔗
      \begin{equation*} (1 - 2)(1 - 4) = (-1)(-3) = 3 \gt 0 \quad \text{(positive)} \end{equation*}
      🔗
    • For \(2 \lt x \lt 4\text{,}\) choose \(x = 3\text{:}\)
      🔗
      \begin{equation*} (3 - 2)(3 - 4) = (1)(-1) = -1 \lt 0 \quad \text{(negative)} \end{equation*}
      🔗
    • For \(x \gt 4\text{,}\) choose \(x = 5\text{:}\)
      🔗
      \begin{equation*} (5 - 2)(5 - 4) = (3)(1) = 3 \gt 0 \quad \text{(positive)} \end{equation*}
      🔗
    To illustrate these results, we can put a \(+\) or \(-\) above each interval, depending if it’s positive or negative.
    🔗
    This means that the solution is \(x \lt 2\) or \(x \gt 4\text{.}\) We can denote this graphically by highlighting those regions.
    🔗
    We use open circles (basically, hollow circles) to indicate that \(x=2\) and \(x=4\) are not included in the solution, because the inequality is \(\lt\text{.}\)
    🔗
    In fact, when doing test points, we only need to know the sign of the output (whether it is positive or negative). Its exact value as a number isn’t necessary. Recall that with multiplying positive and negative numbers,
    🔗
    \begin{align*} (+)(+) \amp= +\\ (+)(-) \amp= - \quad \text{and} \quad (-)(+) = -\\ (-)(-) \amp= + \end{align*}
    Then, testing points could look like this,
    🔗
    • Test \(x = 1\text{:}\) \((1 - 2)(1 - 4) = (-)(-) = +\)
      🔗
      🔗
    • Test \(x = 3\text{:}\) \((3 - 2)(3 - 4) = (+)(-) = -\)
      🔗
      🔗
    • Test \(x = 5\text{:}\) \((5 - 2)(5 - 4) = (+)(+) = +\)
      🔗
      🔗
    🔗
    • The roots of the parabola are called critical points, because they are the important points that indicate where the parabola can change sign from positive to negative, or vice versa.
      🔗
      🔗
    • The number line with labelled \(+\) and \(-\) intervals is called a sign chart, and this method is sometimes called the test point method.
      🔗
      🔗
    In summary, after finding the critical values,
    🔗
    • Choose a test point in each interval, and evaluate the quadratic function there to determine if it is positive or negative, i.e. if that interval is in the solution set.
      🔗
      🔗

    Remark 3.10.6.

    Some teachers require you to use a more elaborate table to do a sign chart. I personally believe this chart is excessively tedious and doesn’t add much, but some teachers require it to properly show your work.
    🔗
    \begin{equation*} \begin{array}{c|c|c|c} \amp x\lt 2 \amp 2\lt x\lt 4 \amp x\gt 4 \\ \hline x-2 \amp - \amp + \amp + \\ x-4 \amp - \amp - \amp + \\ \hline (x-2)(x-4) \amp + \amp - \amp + \end{array} \end{equation*}
    🔗
    🔗

    Subsection 3.10.4 Examples

    Example 3.10.7. Solve \(x^2 - 4 \geq 0\).

    First, find the critical values by solving the corresponding equation,
    🔗
    \begin{align*} x^2 - 4 \amp= 0\\ x^2 \amp= 4\\ x \amp= \pm 2 \end{align*}
    So, the critical values are \(x = 2\) and \(x = -2\text{.}\) These divide the number line into three intervals. Since the leading coefficient is positive (\(+1\)), the parabola opens up, so the sign pattern is \(+ \, - \, +\text{.}\)
    🔗
    We want where the expression is greater than or equal to zero, which is the positive parts, so it’s when \(x \leq -2\) or when \(x \geq 2\text{.}\) We include the endpoints, because the inequality includes equals. Therefore, \(x \leq -2\) or \(x \geq 2\text{.}\)
    🔗
    🔗

    Example 3.10.8. Solve \(9 - x^2 \geq 0\).

    The goal is to find all values of \(x\) such that \(9 - x^2 \geq 0\text{.}\) First, find the critical values by solving the corresponding equation,
    🔗
    \begin{align*} 9 - x^2 \amp= 0\\ 9 \amp= x^2\\ x \amp= \pm 3 \end{align*}
    So, the critical values are \(x = 3\) and \(x = -3\text{.}\) These divide the number line into three intervals. Notice that the leading coefficient is negative (\(-1\text{,}\) from the \(-x^2\) term), so the parabola opens down. This means the sign pattern is \(- \, + \, -\text{.}\)
    🔗
    We want where the expression is greater than or equal to zero, which is the positive part, and so when \(x\) is between \(-3\) and 3. We also include the endpoints, because the inequality is \(\geq\text{.}\) Therefore, the solution is \(-3 \leq x \leq 3\text{.}\)
    🔗
    🔗

    Exercise Group 3.10.6. Creating an Inequality Given Its Solution.

    Write a quadratic inequality with the given solution, in standard form.
    🔗
    (a)
    \(-1 \lt x \lt 2\)
    Answer.

    🔗
    🔗
    🔗
    (b)
    \([-\frac{1}{2}, 5]\)
    Answer.

    🔗
    🔗
    🔗
    (c)
    \(x \leq -4\) or \(x \geq 3\)
    Answer.

    🔗
    🔗
    🔗

    Example 3.10.10. Projectile Motion.

    A ball is thrown upward at 15 m/s from a height of 2 m. Its height \(h\) at time \(t\) seconds after being thrown, can be modelled by the function \(h(t) = -4.9t^2 + 15t + 2\text{.}\) Find how long the ball was in the air for before it hit the ground, rounded to the nearest tenth of a second.
    🔗
    Answer.
    3.2 seconds.
