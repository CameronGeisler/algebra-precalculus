    Section 2.6 Quadratic Inequalities in One Variable

    In addition to quadratic equations, we can also consider quadratic inequalities, which are related.
    🔗

    Example 2.6.1. Motivation: graphical method.

    Consider the inequality,
    🔗
    \begin{gather*} x^2-4x+3 \gt 0 \end{gather*}
    The goal is to find all of the values of \(x\) such that \(x^2 - 4x + 3 \gt 0\text{.}\) In simple terms, this means all numbers \(x\text{,}\) where if you substitute it into \(x^2-4x+3\text{,}\) the result is positive. This is most intuitive by visualizing the graph of the quadratic function \(y = x^2 - 4x + 3\text{,}\)
    🔗
    Then, determining where \(x^2-4x+3 \gt 0\) is equivalent to determining the \(x\) where the \(y\)-coordinate of the parabola is positive. Observe that the \(x\)-intercepts of the graph are \(x = 1, 3\text{.}\) So, the graph is positive when \(x \lt 1\) or when \(x \gt 3\text{.}\) Therefore, the solution is \(x \lt 1\) or \(x \gt 3\text{.}\)
    🔗
    🔗

    Example 2.6.2. Motivation: sign analysis with roots.

    Consider the inequality \(x^2-x-6 \lt 0\text{.}\) Like before, first, find the roots of the parabola, by making it an equals sign, and factoring,
    🔗
    \begin{align*} x^2 - x - 6 \amp = 0\\ (x-3)(x+2) \amp = 0 \end{align*}
    So, the roots are \(x = 3\) and \(x = -2\text{.}\) Since \(x^2-x-6\) has a positive leading coefficient (+1), the parabola opens up. This means that the quadratic roughly looks like this,
    🔗
    From this, the parabola is negative when \(x\) is between \(-2\) and \(3\text{,}\) or \(-2 \lt x \lt 3\text{.}\)
    🔗
    🔗
    To solve the inequality, we don’t need to graph the entire parabola precisely. Instead, we only need to find the roots, and know the basic shape (opening up or down).
    🔗
    A quadratic expression can only change sign at its roots. This means, if a quadratic has two roots, then:
    🔗
    • If the quadratic is positive in one interval, it will switch to negative when crossing a root.
      🔗
      🔗
    • After crossing the second root, it switches back to positive.
      🔗
      🔗
    If the quadratic opens down, then the pattern is reversed: it first is negative, then positive, then negative. The pattern of signs follows,
    🔗
    \begin{gather*} \text{either} \quad + \, - \, + \quad \text{or} \quad - \, + \, - \end{gather*}

    Subsection 2.6.2 Solving Quadratic Inequalities Summary

    Definition 2.6.4.

    A quadratic inequality is an inequality that can be written in one of the forms,
    🔗
    \begin{align*} ax^2 + bx + c \amp \leq 0\\ ax^2 + bx + c \amp \lt 0\\ ax^2 + bx + c \amp \geq 0\\ ax^2 + bx + c \amp \gt 0 \end{align*}
    🔗
    1. (If necessary) Arrange the quadratic inequality in standard form, with \(ax^2 + bx + c\) on one side, and 0 on the other.
      🔗
      🔗
    2. Find the critical values (the roots of the quadratic expression), by solving the corresponding quadratic equation \(ax^2 + bx + c = 0\text{.}\)
      🔗
      🔗
    3. Sketch a number line, and mark the critical values to split it into 3 intervals.
      🔗
      🔗
    4. Find where the quadratic is positive or negative
      🔗
      • If it opens up (\(a \gt 0\)), the sign pattern is \(= \, - \, +\text{.}\)
        🔗
        🔗
      • If it opens down (\(a \lt 0\)), the sign pattern is \(- \, + \, -\text{.}\)
        🔗
        🔗
      🔗
    5. Choose the intervals that match the inequality symbol (greater than or less than).
      🔗
      🔗
    6. Include the endpoints if the inequality is \(\geq\) or \(\leq\) (non-strict inequality). Otherwise, don’t include the endpoints if it’s \(\gt\) or \(\lt\text{.}\)
      🔗
      🔗
    🔗
    🔗
