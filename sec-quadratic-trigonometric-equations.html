    Section 6.3 Quadratic Trigonometric Equations

    Some trigonometric equations are quadratic in a particular trigonometric function, in that they involve a trigonometric function being squared. In this way, many methods for solving quadratic equations can be used to solve trig equations.
    🔗

    Subsection 6.3.1 Notation \(\sin^2{x}\) and Other Powers of a Trigonometric Function

    Sometimes, we want to write the square of a trig function. For example, \(\sin{x} \cdot \sin{x}\text{,}\) or \(\brac{\sin{x}}^2\text{.}\) It is common to write this more simply as \(\sin^2{x}\text{.}\) For example,
    🔗
    \begin{gather*} \sin^2{\brac{\frac{\pi}{6}}} = \brac{\sin{\brac{\frac{\pi}{6}}}}^2 = \brac{\frac{1}{2}}^2 = \frac{1}{4} \end{gather*}
    This is a convention that saves space and avoids having to write so many parentheses.
    🔗
    This notation also applies generally to all trigonometric functions,
    🔗
    \begin{align*} \boxed{\begin{array}{rl} \sin^2{x} \amp = (\sin{x})^2 \\ \cos^2{x} \amp = (\cos{x})^2 \\ \tan^2{x} \amp = (\tan{x})^2 \end{array}} \qquad \text{and so on, for the other trig functions.} \end{align*}
    Also, this works the same with higher powers (e.g. \(\tan^3{x} = (\tan{x})^3\)).
    🔗
    This notation often confuses students when first learning it. In particular, the exponent (2 in this case) applies to the entire trigonometric function, not the angle \(x\text{.}\) In other words,
    🔗
    \begin{gather*} \underbrace{\sin^2{x}}_{\text{Sine of } x \text{, then squared}} \neq \underbrace{\sin{\brac{x^2}}}_{\text{Take the sine of } x^2} \end{gather*}
    If you like, whenever you see a squared trig function, you can first write it with brackets, before proceeding with the problem.
    🔗
    🔗

    Subsection 6.3.2 Solving Quadratic Trigonometric Equations

    Example 6.3.1.

    Consider \(2\sin^2{x} - 1 = 0\text{.}\) To solve this, you can replace \(\sin{x}\) with a single variable \(u\) (\(u = \sin{x}\)), to get \(2u^2-1=0\text{.}\)
    🔗
    🔗
    More precisely, if \(p(x) = 0\) is a quadratic equation, then the equation can be put into the form \(p(\sin{x}) = 0\) (or another trigonometric function).
    🔗
    🔗
