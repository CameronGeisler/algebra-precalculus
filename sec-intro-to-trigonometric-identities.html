    Section 14.1 Intro to Trigonometric Identities

    In algebra, there are various equations which are always true, called identities.
    🔗

    Example 14.1.1.

    For example,
    🔗
    \begin{gather*} (x + y)^2 = x^2 + 2xy + y^2 \end{gather*}
    These two sides are exactly the same, for any value of \(x\) and \(y\text{.}\) This is called an identity.
    🔗
    🔗
    In a similar way, we can consider a trigonometric identity, a trigonometric equation that is true for all values of the variable.
    🔗
    We already know some basic identities, which could almost be called definitions.
    🔗
    \begin{align*} \amp \boxed{\begin{gathered}\textbf{Reciprocal identities}\\[2pt]\csc\theta=\dfrac{1}{\sin\theta}\qquad\sec\theta=\dfrac{1}{\cos\theta}\qquad\cot\theta=\dfrac{1}{\tan\theta}\end{gathered}}\\[8pt]\\ \amp \boxed{\begin{gathered}\textbf{Quotient identities}\\[2pt]\tan\theta=\dfrac{\sin\theta}{\cos\theta}\qquad\cot\theta=\dfrac{\cos\theta}{\sin\theta}\end{gathered}} \end{align*}

    Subsection 14.1.1 Pythagorean Identities

    There is also the Pythagorean identity, which comes from the unit circle,
    🔗
    \begin{gather*} \boxed{\sin^2{x} + \cos^2{x} = 1} \qquad \text{for all $x$} \end{gather*}
    Equivalently,
    🔗
    \begin{align*} \sin^2{x} \amp = 1 - \cos^2{x} \\\\ \cos^2{x} \amp = 1 - \sin^2{x} \end{align*}
    This identity can be used to derive further identities,
    🔗
    \begin{gather*} \boxed{1 + \tan^2{x} = \sec^2{x}} \qquad \text{and} \qquad \boxed{1 + \cot^2{x} = \csc^2{x}} \end{gather*}
    At first, you should solve problems with these identities as a reference. With practice, you will naturally remember them.
    🔗
    🔗

    Subsection 14.1.2 Simplifying Trigonometric Expressions

    Exercise Group 14.1.1. Simplifying Expressions.

    Simplify each trigonometric expression.
    🔗
    (a)
    \(\sec{x} \sin{x}\)
    Hint.
    quotient identity, cancel.
    🔗
    🔗
    Answer.
    \(\tan{x}\)
    🔗
    🔗
    (b)
    \((1 - \tan{x})(1 + \cot{x})\)
    Hint.
    expand.
    🔗
    🔗
    Answer.
    \(\cot{x} - \tan{x}\)
    🔗
    🔗
    (c)
    \(\dfrac{\sec{x}}{\tan{x}}\)
    Hint.
    quotient, cancel.
    🔗
    🔗
    Answer.
    \(\csc{x}\)
    🔗
    🔗
    (d)
    \((\tan{x} + \sec{x})(\tan{x} - \sec{x})\)
    Hint.
    expand, Pythagorean identity.
    🔗
    🔗
    Answer.
    \(-1\)
    🔗
    🔗
    (e)
    \(\cos^4{x} - \sin^4{x}\)
    Hint.
    difference of squares, Pythagorean identity.
    🔗
    🔗
    Answer.
    \(\cos^2{x} - \sin^2{x}\)
    🔗
    🔗
    (f)
    \(\dfrac{\cot^2{x}}{1 - \sin^2{x}}\)
    Hint.
    quotient identity, Pythagorean identity.
    🔗
    🔗
    Answer.
    \(\csc^2{x}\)
    🔗
    🔗
    (g)
    \(\sin{x} \cos{x} (\csc{x} + \sec{x})\)
    Hint.
    distribute, quotient identity, cancel.
    🔗
    🔗
    Answer.
    \(\cos{x} + \sin{x}\)
    🔗
    🔗
    (h)
    \((\tan{x} + \cot{x})^2 - (\tan{x} - \cot{x})^2\)
    Hint.
    ideally factor as a difference of squares, if not then expand and collect like terms.
    🔗
    🔗
    Answer.
    \(4\)
    🔗
    🔗
    (i)
    \(\dfrac{1}{\cos{x} + 1} - \dfrac{1}{\cos{x} - 1}\)
    Hint.
    combine fractions, collect like terms, Pythagorean identity.
    🔗
    🔗
    Answer.
    \(2\csc^2{x}\)
    🔗
    🔗
    (j)
    \(\dfrac{\csc{x} - \sin{x}}{\cot{x}}\)
    Hint.
    reciprocal identity, combine to one fraction, Pythagorean identity, cancel.
    🔗
    🔗
    Answer.
    \(\cos{x}\)
    🔗
    🔗
    (k)
    \(\dfrac{\tan{x} - \tan{x}\sin^2{x}}{\cos^2{x}}\)
    Hint.
    factor out \(\tan{x}\text{,}\) Pythagorean identity, cancel.
    🔗
    🔗
    Answer.
    \(\tan{x}\)
    🔗
    🔗
    🔗

    (a)

    🔗

    🔗
    (b)
    \(\dfrac{\cos^2{x} - 4}{7\cos{x} - 14}\)

    🔗

    🔗
    🔗
    (c)

    Hint.

    🔗
    🔗
    Answer.
    \(\sin{x} - 1\)

    Subsection 14.1.3 Solving Equations Using Identities

    As you learn more fundamental trigonometric identities, you will be able to solve more trigonometric equations algebraically.
    🔗

    Exercise Group 14.1.5. Solve Each Equation.

    Solve each equation, by finding all solutions in the interval \([0,2\pi)\) and the general solution.
    🔗
    (a)
    \(2\cos^2{x} - 3 \sin{x} = 0\)
    Hint.
    Pythagorean identity, leads to \(2\sin^2{x} + 3\sin{x} - 2 = 0\text{,}\) factor, leads to \(\sin{x} = \frac{1}{2}\) or \(\sin{x} = -2\text{.}\)
    🔗
    🔗
    Answer.
    \(x = \frac{\pi}{6}, \frac{5\pi}{6}\text{.}\) General solution: \(x = \frac{\pi}{6} + 2\pi n, x = \frac{5\pi}{6} + 2\pi n, n \in \mathbb{I}\text{.}\)
    🔗
    🔗
    🔗
    (b)
    \(\tan^2{x} - 4\sec{x} + 5 = 0\)
    Hint.
    Pythagorean identity, leads to \(\sec^2{x} - 4\sec{x} + 4 = 0\text{,}\) factor, leads to \(\sec{x} = 2\text{.}\)
    🔗
    🔗
    Answer.
    \(x = \frac{\pi}{3}, \frac{5\pi}{3}\text{.}\) General solution: \(x = \frac{\pi}{3} + 2\pi n, x = \frac{5\pi}{3} + 2\pi n, n \in \mathbb{I}\text{.}\)
    🔗
    🔗
    🔗
    (c)
    \(\cos{x} - 2\sin^2{x} + 1 = 0\)
    Hint.
    Pythagorean identity, leads to \(2\cos^2{x} + \cos{x} - 1 = 0\text{,}\) factor, leads to \(\cos{x} = \frac{1}{2}\) or \(\cos{x} = -1\text{.}\)
    🔗
    🔗
    Answer.
    \(x = \frac{\pi}{3}, \pi, \frac{5\pi}{3}\text{.}\) General solution: \(x = \frac{\pi}{3} + 2\pi n, x = \frac{5\pi}{3} + 2\pi n, x = \pi + 2\pi n, n \in \mathbb{I}\text{.}\)
    🔗
    🔗
    🔗
    (d)
    \(2\cos^2{x} + \tan^2{x} = 2\)
    Hint.
    Pythagorean identity, leads to \(2\cos^4{x} - 3\cos^2{x} + 1 = 0\text{,}\) factor, leads to \(\cos^2{x} = 1\) or \(\cos^2{x} = \frac{1}{2}\text{.}\)
    🔗
    🔗
    Answer.
    \(x = 0, \frac{\pi}{4}, \frac{3\pi}{4}, \pi, \frac{5\pi}{4}, \frac{7\pi}{4}\text{.}\) General solution: \(x = n\pi, x = \frac{\pi}{4} + \frac{\pi}{2}n, n \in \mathbb{I}\text{.}\)
    🔗
    🔗
    🔗
    (e)
    \(\sin^2{x} + \cot^2{x} - 1 = 0\)
    Hint.
    quotient and Pythagorean identity, leads to \(\sin^4{x} - 2\sin^2{x} + 1 = 0\text{,}\) factor, leads to \(\sin{x} = 1\) or \(\sin{x} = -1\text{.}\)
    🔗
    🔗
    Answer.
    \(x = \frac{\pi}{2}, \frac{3\pi}{2}\text{.}\) General solution: \(x = \frac{\pi}{2} + \pi n, n \in \mathbb{I}\text{.}\)
    🔗
    🔗
    🔗
    (f)
    \(\csc^2{x} - \cot{x} - 1 = 0\)
    Hint.
    Pythagorean identity, leads to \(\cot^2{x} - \cot{x} = 0\text{,}\) factor, leads to \(\cot{x} = 0\) or \(\cot{x} = 1\text{.}\)
    🔗
    🔗
    Answer.
    \(x = \frac{\pi}{2}, \frac{3\pi}{2}, \frac{\pi}{4}, \frac{5\pi}{4}\text{.}\) General solution: \(x = \frac{\pi}{2} + \pi n, x = \frac{\pi}{4} + \pi n, n \in \mathbb{I}\text{.}\)
